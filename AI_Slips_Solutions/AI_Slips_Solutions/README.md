# AI Slips Solutions

This directory contains solutions for all 25 Artificial Intelligence (AI) slips.

## Structure
- Each slip has its own folder (Slip1 to Slip25)
- Each folder contains two Python programs (Q1.py and Q2.py)
- Each program includes the original question as a comment

## Execution
Use the provided run_slip.py script:

### Run specific program:
\\\
python run_slip.py 1 1    # Run Slip1 Q1
python run_slip.py 5 2    # Run Slip5 Q2
\\\

### Run all programs:
\\\
python run_slip.py all
\\\

## Requirements
Install required packages:
\\\
pip install -r requirements.txt
\\\

## Notes
- Programs are based on the AI programs PDF
- All code uses standard Python libraries
- Questions are included as comments in each file
- Some programs require NLTK data downloads
